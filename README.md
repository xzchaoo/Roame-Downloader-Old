# Linux cd命令行小工具 #
## 简介 ##
[路游网](http://www.roame.net/)专辑批量下载, 年久失修, 不知道还可用不.

## 基本策略 ##
1. 由于网站没有验证码机制, 因此就是非常简单的模拟HTTP请求就行了.
2. 一个普通账号一次只能下载一张图片, 速度20KB/S左右.
3. 但是一个IP似乎可以开多个不同的账号进行下载, 这样速度就提高了.
4. 注意最后产生的图片的URL有实效性.
5. 只需要到对应的网站去"摸爬滚打"即可.
